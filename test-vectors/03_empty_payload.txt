Vector 03: empty_payload
------------------------
Frame with zero-length payload. Verifies empty-plaintext
sealing works correctly.

Scenario:
source_id = "XENIATST"
epoch     = 0x42
key       = fixture-key
seq       = 0
frame_id = 42, payload = []

Envelope is 12 (nonce) + 24 (bincode header: frame_id + ts + len=0)
+ 16 (tag) = 52 bytes.
