Vector 04: long_payload
-----------------------
Frame with a 256-byte payload containing every possible byte
value 0x00..=0xFF. Verifies sealing on non-compressible,
all-range plaintext.

Scenario:
source_id = "XENIATST"
epoch     = 0x42
key       = fixture-key
seq       = 0
payload = 0x00, 0x01, 0x02, ..., 0xFF (256 bytes)
