Vector 01: hello_frame
----------------------
A minimal Frame with a 12-byte UTF-8 payload "hello, xenia".
This is the canonical roundtrip fixture.

Scenario:
source_id = "XENIATST"  (8 bytes)
epoch     = 0x42
key       = "xenia-wire-test-vector-key-2026!"  (32 bytes)
seq       = 0  (first seal on this session)
payload_type = 0x10 (FRAME)

timestamp_ms = 1700000000000 (u64 LE: 008b0e0095010000)
payload      = "hello, xenia" (length-prefixed: 0c00000000000000 + utf8)
